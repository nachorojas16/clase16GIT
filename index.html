Git y GitHub

Git

  • Es un software
  • Es mantenido por Linux
  • Es un repositorio local
  • Es de código abierto
  • No tiene gestión de usuarios
  • Se basa en un sistema de comandos
  • Control de versiones

GitHub

  • Es un servicio
  • Es mantenido por Microsoft
  • Es un repositorio remoto
  • Es gratuito y pago
  • Tiene incorporado la gestión de usuarios
  • Almacena versiones repositorios de git
